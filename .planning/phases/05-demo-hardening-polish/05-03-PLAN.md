VITE_API_URL pattern (apply to ALL fetch('/api/...') calls):
```typescript
// Before:
await fetch('/api/route', { method: 'POST', body: JSON.stringify(payload) })
// After:
await fetch(`${import.meta.env.VITE_API_URL ?? ''}/api/route`, { method: 'POST', body: JSON.stringify(payload) })
```
Search command to find all fetch calls: grep -r "fetch('/api\|fetch(\`/api" src/

Server-down error dispatch (in catch block of POST /api/route):
```typescript
try {
  const res = await fetch(`${import.meta.env.VITE_API_URL ?? ''}/api/route`, { ... });
  if (!res.ok) throw new Error('route-failed');
  // handle success...
} catch {
  useAppStore.getState().setSimulationStatus('error');
}
```
